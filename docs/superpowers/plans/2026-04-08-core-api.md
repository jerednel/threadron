# AgentTask Core API — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the core REST API, database, Docker Compose stack, and skill.md so that agents can create, manage, and query tasks via HTTP — and self-hosters can run it with `docker compose up -d`.

**Architecture:** TypeScript API server (Hono framework) backed by PostgreSQL (Drizzle ORM). Stateless, horizontally scalable. Containerized with Docker Compose (3 containers: API, DB, web placeholder). All state in Postgres.

**Tech Stack:** TypeScript, Node.js 22, Hono, Drizzle ORM, PostgreSQL 16, Vitest, Docker, Docker Compose

**Scope:** Core API only (Phase 1). Web dashboard is Phase 2 (separate plan). Smart Layer is out of scope for v1.

Create `src/middleware/auth.ts`:

```typescript
import { createMiddleware } from "hono/factory";
import { eq } from "drizzle-orm";
import type { PostgresJsDatabase } from "drizzle-orm/postgres-js";
import * as schema from "../db/schema.js";

export function authMiddleware(db: PostgresJsDatabase<typeof schema>) {
  return createMiddleware(async (c, next) => {
    const header = c.req.header("Authorization");
    if (!header?.startsWith("Bearer ")) {
      return c.json({ error: "Missing or invalid Authorization header" }, 401);
    }

    const key = header.slice(7);
    const [row] = await db
      .select()
      .from(schema.apiKeys)
      .where(eq(schema.apiKeys.key, key))
      .limit(1);

    if (!row) {
      return c.json({ error: "Invalid API key" }, 401);
    }

    // Update agent's last_seen if this key is linked to an agent
    if (row.agentId) {
      await db
        .update(schema.agents)
        .set({ lastSeen: new Date() })
        .where(eq(schema.agents.id, row.agentId));
    }

    c.set("apiKey", row);
    await next();
  });
}
```

- [ ] **Step 4: Implement auth routes**

Create `src/routes/auth.ts`:

```typescript
import { Hono } from "hono";
import { eq } from "drizzle-orm";
import type { PostgresJsDatabase } from "drizzle-orm/postgres-js";
import * as schema from "../db/schema.js";
import { genId } from "../lib/id.js";

export function authRoutes(db: PostgresJsDatabase<typeof schema>) {
  const router = new Hono();

  // POST /setup — initial setup (first run only)
  router.post("/setup", async (c) => {
    const existing = await db.select().from(schema.apiKeys).limit(1);
    if (existing.length > 0) {
      return c.json({ error: "Setup already completed. Use POST /v1/auth/keys to create additional keys." }, 409);
    }

    const body = await c.req.json<{ name: string }>();
    const key = `tfa_sk_${genId("k")}`;
    const id = genId("key");

    const [row] = await db
      .insert(schema.apiKeys)
      .values({ id, key, name: body.name })
      .returning();

    return c.json({ id: row.id, api_key: row.key, name: row.name }, 201);
  });

  // POST /keys — create a new API key (requires auth)
  router.post("/keys", async (c) => {
    const body = await c.req.json<{ name: string; agent_id?: string }>();
    const key = `tfa_sk_${genId("k")}`;
    const id = genId("key");

    const [row] = await db
      .insert(schema.apiKeys)
      .values({ id, key, name: body.name, agentId: body.agent_id || null })
      .returning();

    return c.json({ id: row.id, api_key: row.key, name: row.name, agent_id: row.agentId }, 201);
  });

  // GET /keys — list API keys (redacted)
  router.get("/keys", async (c) => {
    const rows = await db.select().from(schema.apiKeys);
    const redacted = rows.map((r) => ({
      id: r.id,
      name: r.name,
      agent_id: r.agentId,
      key_prefix: r.key.slice(0, 12) + "...",
      created_at: r.createdAt,
    }));
    return c.json({ keys: redacted });
  });

  // DELETE /keys/:id — revoke API key
  router.delete("/keys/:id", async (c) => {
    const id = c.req.param("id");
    const [deleted] = await db
      .delete(schema.apiKeys)
      .where(eq(schema.apiKeys.id, id))
      .returning();

    if (!deleted) {
      return c.json({ error: "API key not found" }, 404);
    }
    return c.json({ deleted: true });
  });

  return router;
}
```

export function taskRoutes(db: PostgresJsDatabase<typeof schema>) {

  router.post("/", async (c) => {

      title: string;
      domain_id: string;
      created_by: string;
      project_id?: string;
      assignee?: string;
      priority?: string;
      guardrail?: string;
      dependencies?: string[];

        title: body.title,
        domainId: body.domain_id,
        projectId: body.project_id || null,
        createdBy: body.created_by,
        assignee: body.assignee || null,
        priority: body.priority || "medium",
        guardrail: body.guardrail || "autonomous",
        dependencies: body.dependencies || [],
        dueDate: body.due_date ? new Date(body.due_date) : null,
        tags: body.tags || [],
        metadata: body.metadata || {},
      })
      .returning();

    return c.json(toApi(row), 201);

import type { PostgresJsDatabase } from "drizzle-orm/postgres-js";
import * as schema from "../db/schema.js";

function toApi(row: typeof schema.agents.$inferSelect) {
  return {
    id: row.id,
    name: row.name,
    type: row.type,
    capabilities: row.capabilities,
    last_seen: row.lastSeen,
    created_at: row.createdAt,
  };
}

export function agentRoutes(db: PostgresJsDatabase<typeof schema>) {
  const router = new Hono();

  router.post("/", async (c) => {
    const body = await c.req.json<{
      id: string;
      name: string;
      type?: string;
      capabilities?: string;
    }>();

    const [row] = await db
      .insert(schema.agents)
      .values({
        id: body.id,
        name: body.name,
        type: body.type || "custom",
        capabilities: body.capabilities || null,
      })
      .returning();

    return c.json(toApi(row), 201);
  });

  router.get("/", async (c) => {
    const rows = await db.select().from(schema.agents);
    return c.json({ agents: rows.map(toApi) });
  });

  router.get("/:id", async (c) => {
    const id = c.req.param("id");
    const [row] = await db
      .select()
      .from(schema.agents)
      .where(eq(schema.agents.id, id))
      .limit(1);

    if (!row) return c.json({ error: "Agent not found" }, 404);
    return c.json(toApi(row));
  });

  router.patch("/:id", async (c) => {
    const id = c.req.param("id");
    const body = await c.req.json<{
      name?: string;
      type?: string;
      capabilities?: string;
    }>();

    const updates: Record<string, unknown> = {};
    if (body.name !== undefined) updates.name = body.name;
    if (body.type !== undefined) updates.type = body.type;
    if (body.capabilities !== undefined) updates.capabilities = body.capabilities;

    const [row] = await db
      .update(schema.agents)
      .set(updates)
      .where(eq(schema.agents.id, id))
      .returning();

    if (!row) return c.json({ error: "Agent not found" }, 404);
    return c.json(toApi(row));
  });

  return router;
}

import { eq } from "drizzle-orm";
import type { PostgresJsDatabase } from "drizzle-orm/postgres-js";
import * as schema from "../db/schema.js";

function toApi(row: typeof schema.config.$inferSelect) {
  return {
    key: row.key,
    value: row.value,
    updated_at: row.updatedAt,
  };
}

export function configRoutes(db: PostgresJsDatabase<typeof schema>) {
  const router = new Hono();

  router.post("/", async (c) => {
    const body = await c.req.json<{ key: string; value: unknown }>();

    const [row] = await db
      .insert(schema.config)
      .values({ key: body.key, value: body.value })
      .onConflictDoUpdate({
        target: schema.config.key,
        set: { value: body.value, updatedAt: new Date() },
      })
      .returning();

    return c.json(toApi(row), 201);
  });

  router.get("/", async (c) => {
    const rows = await db.select().from(schema.config);
    return c.json({ config: rows.map(toApi) });
  });

  router.get("/:key", async (c) => {
    const key = c.req.param("key");
    const [row] = await db
      .select()
      .from(schema.config)
      .where(eq(schema.config.key, key))
      .limit(1);

    if (!row) return c.json({ error: "Config key not found" }, 404);
    return c.json(toApi(row));
  });

  router.patch("/:key", async (c) => {
    const key = c.req.param("key");
    const body = await c.req.json<{ value: unknown }>();

    const [row] = await db
      .update(schema.config)
      .set({ value: body.value, updatedAt: new Date() })
      .where(eq(schema.config.key, key))
      .returning();

    if (!row) return c.json({ error: "Config key not found" }, 404);
    return c.json(toApi(row));
  });

  return router;
}

If no duplicates found, create the task:

```
POST {TFA_API_URL}/v1/tasks
Authorization: Bearer {TFA_API_KEY}
Content-Type: application/json

{
  "title": "Short, clear description of what needs to be done",
  "domain_id": "{domain_id}",
  "project_id": "{project_id or omit}",
  "created_by": "{TFA_AGENT_ID}",
  "assignee": "{TFA_AGENT_ID or another agent}",
  "priority": "medium",
  "guardrail": "autonomous",
  "tags": ["optional", "tags"],
  "metadata": {"source": "jira", "jira_id": "PROJ-123"}
}
```

Use the domain's default guardrail level unless the task specifically warrants a different level.

### Working on Tasks

1. Claim the task:
```
PATCH {TFA_API_URL}/v1/tasks/{task_id}
Authorization: Bearer {TFA_API_KEY}
Content-Type: application/json

{"status": "in_progress"}
```

2. Add context as you work:
```
POST {TFA_API_URL}/v1/tasks/{task_id}/context
Authorization: Bearer {TFA_API_KEY}
Content-Type: application/json

{
  "type": "observation|decision|blocker|progress|artifact",
  "body": "Describe what you found, decided, or produced",
  "author": "{TFA_AGENT_ID}"
}
```

Context types:
- **observation** — Something you noticed or learned ("Root cause is X")
- **decision** — A choice you made and why ("Using approach A because...")
- **blocker** — Something preventing progress ("Need API key for service X")
- **progress** — Status update ("Tests passing, deploying to staging")
- **artifact** — A deliverable ("PR #42 created", "Report at /path/to/file")

3. Complete the task:
```
PATCH {TFA_API_URL}/v1/tasks/{task_id}
Authorization: Bearer {TFA_API_KEY}
Content-Type: application/json

{"status": "completed"}
```

Always add a final context entry summarizing what was done before marking complete.

### Handling Guardrails

- **autonomous**: Act freely. Log what you did via context entries.
- **notify**: Act, but add a context entry of type `progress` so the dashboard shows activity.
- **approval_required**: Present your plan to the user. Do NOT proceed until they approve. Add a context entry of type `decision` with the approved plan.

### Respecting Other Agents

- Read the full context log before working on a task another agent started: `GET {TFA_API_URL}/v1/tasks/{task_id}/context`
- Don't reassign tasks owned by another agent without user approval
- When creating tasks, check if a similar task already exists (use search)
- If you notice overlapping work, flag it to the user

## Re-Onboarding

When the user says "redo onboarding", "change preferences", "update domains", or similar:

1. Read current config: `GET {TFA_API_URL}/v1/config`
2. Read current domains: `GET {TFA_API_URL}/v1/domains`
3. Present current settings to the user
4. Ask what they want to change
5. Update via the API (PATCH for existing, POST for new)
6. Existing tasks and context are NOT affected — only defaults for future tasks change

Content-Type: application/json
