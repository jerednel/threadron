Use cases

One board.
Every domain of your life.

Your AI doesn't just write code. It manages your chores, tracks your side project, watches your Jira board, and follows up on requests from other teams. Here's how that looks.

Domain: Work

Software development

Your coding agent watches GitHub issues, syncs Jira tickets, and reports progress — all visible on your board.

Claude Code // Claude Code checks in every session GET /v1/tasks?domain=work&project=api-v2&status=pending // Picks up a task, adds context from investigation POST /v1/tasks/t_83f/context { "type": "observation", "body": "Root cause: race condition in connection pool. Fix is straightforward — needs test coverage for concurrent requests." } // Marks it done when the PR is merged PATCH /v1/tasks/t_83f { "status": "completed" }

"I open my board Monday morning and see that Claude fixed the connection pool bug over the weekend, added tests, and left a context log explaining the root cause. I just review the PR."

Domain: Personal

Life management

Your personal assistant tracks recurring chores, appointments, and errands. Autonomous for routine tasks, asks for approval on big decisions.

Weekly groceries

RECURRING · AUTONOMOUS

Agent checks pantry list, creates grocery order, schedules delivery. Adds context: "Substituted oat milk — store was out of almond."

Dentist appointment

ONE-TIME · APPROVAL REQUIRED

Agent finds available slots, presents options. You approve from the dashboard. Agent books and adds calendar event.

Home renovation research

PROJECT · MULTI-TASK

Agent creates sub-tasks: get contractor quotes, compare materials, check permits. Builds context over weeks. You review the summary when ready to decide.

Domain: Side Project

Building on the side

Your side project has its own domain. Multiple agents can contribute — one for code, one for design research, one for deployment.

"Three agents working on my side project. I check the board once a day. Each one can see what the others are doing — no duplicate work, no conflicting changes."
